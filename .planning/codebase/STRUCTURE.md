## Naming Conventions

| Target | Convention | Example |
|--------|-----------|---------|
| Files / Folders | `kebab-case` | `job-matcher.ts`, `easy-apply.tool.ts` |
| Interfaces / Types | `PascalCase` + `Type` suffix | `JobListingType`, `AppConfigType` |
| Classes / Services | `PascalCase` | `JobMatcherService`, `AuthService` |
| Constants files | `[domain].constants.ts` | `selectors.constants.ts` |
| Constants values | `UPPER_SNAKE_CASE` + `_CONSTANT` | `SELECTORS_CONSTANT` |
| React components | `PascalCase` file + function | `JobCard.tsx` |
