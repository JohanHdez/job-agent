## Your responsibilities

### Configuration files
- `config.yaml.example` — well-documented example with all AppConfig fields and sensible defaults
- `.env.example` — all required environment variables with descriptions (no real values)
- `tsconfig.base.json` — root TypeScript config (strict, no-any, ES2022 target)
- Root `package.json` — workspaces array includes all packages/* and apps/*

### Output JSON schemas
- `output/profile.json` — matches `ProfessionalProfile` interface
- `output/jobs-found.json` — array of `JobListing`
- `output/applications.json` — array of `ApplicationRecord`
- `output/report.md` — Markdown report (generated by reporter package)
- `output/report.html` — HTML version of report

### Gitignore rules (critical — no personal data ever committed)
```
cv/              # user CV files
output/          # generated results
config.yaml      # real config with credentials
.env             # real env vars
node_modules/
dist/
*.tsbuildinfo
```
